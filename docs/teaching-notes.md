# Teaching Notes

このリポジトリは、最近のコーディング支援ツールに慣れるための小さな題材です。最初から完成品を目指さず、理解しやすい単位で変更していきます。

## 見てほしいポイント

- UI とゲームルールを別ファイルに分ける
- ペットの状態オブジェクトを中心にデータの形を決める
- ルール変更は `logic.test.js` で確認する
- 保存は `localStorage` に閉じ込めて、他のコードから直接触らない

## 進め方

1. まず自分でゲームを動かす
2. 1 つの小さい変更を決める
3. Codex に実装案を出してもらう
4. 差分を読む
5. テストとビルドを実行する
