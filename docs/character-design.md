# Character Design

たまごっち風の育成感は残しつつ、キャラクターはゆるかわ宇宙人系で統一します。どの段階でも、丸い目、アンテナ、宇宙ポッドっぽい輪郭を共通モチーフにします。

## Growth Stages

- `egg`: 宇宙ポッド風のたまご。ヒビやアンテナの動きで mood を出す。
- `baby`: 小さな丸い宇宙人。殻のかけらを身につけて、生まれたばかりの印象にする。
- `kid`: 手足とアンテナが伸びた相棒。好奇心が出るようにポーズを大きくする。
- `teen`: 少し背が伸びたジュニア。襟やホバーブーツで成長感を出す。
- `adult`: 安定した完成形。軌道リングと大きなアンテナで成熟した宇宙ペットにする。

## Mood Rules

- `happy`: 浮く、両手を上げる、アンテナが跳ねる。
- `okay`: 正面で落ち着いた通常ポーズ。
- `tired`: 座る、前傾する、半目、アンテナが垂れる。
- `messy`: 体が傾く、汚れマーク、困り顔。
- `hungry`: お腹を押さえる、口を開ける、食べ物アイコンを見上げる。

## Asset Rules

- SVG は `public/pets/{stage}-{mood}.svg` に置く。
- すべて `256x256 viewBox` で作る。
- 太めのアウトライン、淡い宇宙グリーン、暖色アクセントを基本にする。
