# Roadmap

## 1. まず読む

- `src/main.js`: アプリの起動とイベントの流れ
- `src/game/logic.js`: ペットの状態が変わるルール
- `src/ui/render.js`: 画面を HTML として描く場所

## 2. 小さく変える

- ボタンの文言を変える
- ステータスの増減量を変える
- 成長に必要な時間を変える

## 3. 機能を足す

- 名前入力
- ペット画像の差し替え
- 状態ログ
- 音やアニメーション
- スマホ向けの操作改善

## 4. Codex と練習する相談例

```text
src/game/logic.js を読んで、ペットの成長ルールを説明して。
```

```text
名前を変更できるフォームを追加して。既存の構成に合わせて実装して。
```

```text
空腹度が 0 になったら sick 状態になるようにして、テストも追加して。
```
